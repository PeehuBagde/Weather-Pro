# Weather PRO
Experience weather like never before with real-time data, beautiful visuals, and precise forecasts for any location worldwide.

A modern and responsive weather web application built with React, Vite, and Tailwind CSS.
It allows users to search for real-time weather information of any city with a clean and minimal UI.


🚀 FEATURES

🔍 Search weather by city name

🌡️ Displays temperature, weather condition, humidity, and wind speed

⚡ Fast performance using Vite

🎨 Responsive & modern UI with Tailwind CSS


🛠️ TECH STACK

->Frontend

  React
  
  Vite
  
->Styling

  Tailwind CSS
  
->Tools

  ESLint
  
  Lucide Icons
  
->API

  Weather API (used to fetch real-time weather data)
